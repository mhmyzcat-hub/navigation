# 开款推荐图片

图片请分别放入以下文件夹：

- `suit/`：单西
- `vest-tailcoat/`：单马甲
- `suit-set/`：两件套
- `three-piece/`：三件套
- `trousers/length/`：西裤－裤长
- `trousers/fit/`：西裤－裤型
- `trousers/fabric/`：西裤－面料
- `trousers/pattern/`：西裤－纹样
- `shirt/`：衬衫

图片放入文件夹并推送到 GitHub 后，网页会自动读取，无需登记文件名或修改代码。

分类标题、目录以及是否显示推款联系人统一在根目录的 `data/navigation.js` 中维护。

建议使用容易识别的名称，例如：

- `01-黑色商务西装.jpg`
- `02-灰色休闲西装.jpg`

网页会按照文件名顺序排列。开头的数字序号只用于控制顺序，不会显示在图片标题中；例如 `01-黑色商务西装.jpg` 在网页中显示为“黑色商务西装”。

需要标记为稀缺款时，在文件名中加入 `【稀缺】`：

- `03-双排扣西装【稀缺】.jpg`

网页会显示标题“双排扣西装”，并在标题旁自动添加“·稀缺”标签。

如果只是更新图片，保持文件名不变，直接覆盖原图片再推送即可。
